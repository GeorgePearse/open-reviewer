## Prerequisites

Before starting the tutorials, ensure you have:

- Python 3.12 or later
- An API key from at least one of:
    - [Anthropic](https://console.anthropic.com/) for single-model evaluation
    - [OpenRouter](https://openrouter.ai/) for multi-model consensus
- Basic familiarity with Python and pytest
